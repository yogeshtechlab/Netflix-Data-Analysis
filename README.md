# 🎬 Netflix Data Analysis

## 📌 Project Overview

This project analyzes the Netflix dataset using Python, Pandas, and Matplotlib to uncover trends and insights about movies and TV shows available on the platform.

The analysis focuses on content distribution, ratings, countries, and release year trends to better understand Netflix's content library.

---

## 🎯 Objectives

- Analyze Movies vs TV Shows distribution
- Identify top content-producing countries
- Explore content ratings and classifications
- Study release year trends
- Generate meaningful business insights

---

## 🛠 Technologies Used

- Python
- Pandas
- Matplotlib
- CSV Dataset
- Git & GitHub

---

## 📊 Analysis Performed

### 1. Movies vs TV Shows
- Compared the number of Movies and TV Shows available on Netflix.

### 2. Country Analysis
- Identified the countries contributing the most content.

### 3. Content Ratings Analysis
- Analyzed the most common audience ratings.

### 4. Release Year Trends
- Examined how Netflix content has grown over time.

### 5. Data Cleaning
- Checked and handled missing values.
- Explored dataset structure and quality.

---

## 📈 Key Insights

- Movies dominate the Netflix catalog compared to TV Shows.
- The United States contributes the highest amount of content.
- TV-MA is one of the most common content ratings.
- Netflix content production increased significantly after 2015.
- A small number of countries contribute the majority of content.
